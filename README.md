# TINY5_RISER_C
Based on a-little-wifi's Tinyriser, another riser for Lenovo 8th/9th gen Tiny5 PCs. Exposes the x8 and x4 PCIe links in the riser slot, in the form of a PCIe x8 slot and a PCIe x4 slot.
The project is released in JLC's website.
